===============================================================================
                          GESTIÓN DE LIBROS EN GO
===============================================================================

                               ¡Bienvenido al proyecto!
                  Sistema eficiente y modular para gestionar libros.

===============================================================================
                               FUNCIONALIDAD PRINCIPAL
===============================================================================

- **Agregar un libro**: Permite añadir un nuevo libro al sistema proporcionando 
  los datos del título, autor y año de publicación.
  
- **Listar libros**: Muestra una lista de todos los libros almacenados en el sistema.

- **Eliminar un libro**: Elimina un libro por su título.

- **Buscar libro**: Busca un libro por su título y muestra su información.

- **Manejo de errores**: El sistema gestiona errores de forma clara y coherente, 
  asegurando que cada operación se complete correctamente o mostrando un mensaje 
  de error útil en caso de fallo.

===============================================================================
                               OBJETIVO DEL PROYECTO
===============================================================================

El **Sistema de Gestión de Libros en Go** tiene como objetivo proporcionar una 
plataforma robusta y escalable para la gestión de libros. Con este sistema, podrás
agregar, listar, buscar y eliminar libros, todo gestionado mediante una API RESTful. 
Está diseñado con un enfoque modular, utilizando servicios y un manejo adecuado de 
errores. 

Este proyecto está alojado en un repositorio de GitHub, permitiendo una fácil 
colaboración y control de versiones.
